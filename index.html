USEFUL SQL STATEMENTS

Adapted from TeamTreehouse Instructor, Andrew Chalkley

  1. Return all fields in a table:

    SELECT * FROM movies;

    if your table name is movies.
  2. Return one column from a table:

    SELECT title FROM movies;

  3. Return two or more colunns from a table:

    SELECT title, genre FROM movies;

  4. Return column(s) renamed using AS a table:

    SELECT title AS Title, year AS "Year Released" FROM movies;

  5. Return fields with a condition in a table:

    SELECT first_name, email FROM users WHERE last_name = "Chalkley";

  6. Return fields that involve the inequality operator:

    SELECT * FROM contacts WHERE first_name != "Kenneth";
